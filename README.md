# TripSpark微信小程序

### 项目简介

一个用 Taro框架 + React + TypeScript 开发的微信小程序，为用户提供浏览游记、发布游记、足迹打卡等功能。

### 项目体验

加我联系方式获取二维码

### 项目结构

```
src/
├── api/              # API 请求函数
├── assets/           # 静态资源文件
├── components/       # 通用组件
├── pages/            # 页面组件
├── store/            # 状态管理
├── types/            # TypeScript 类型定义
├── utils/            # 工具函数
│   ├── request.tsx   # 请求封装
│   └── dateFormat.ts # 日期格式化
├── app.config.ts     # 小程序配置文件
├── app.scss          # 全局样式
├── app.ts            # 入口文件
└── custom-variables.scss # 自定义变量
```

### 环境部署

安装依赖: `npm install`
项目开发: `npm run dev:weapp`
项目构建: `npm run build:weapp`
